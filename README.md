# Отразись в Летней школе

Анна Комкова, Анастасия Анисеня

Наша data-art визуализация представлена несколькими частями:

облако слов состоит из ассоциаций, возникающих у людей о Летней школе

буквы ЛШ, покрытые различными узорами, оживают при помощи мэппинга

маленькие забавные анимации и код отображают наше теплое отношение к искусству

Посмотри на свое отражение в Летней школе и почувствуй себя на своём месте!
